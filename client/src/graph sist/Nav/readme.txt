Основной файл с историей:  MarketData.ts
 
Классы: 

 CBar - бар
 CTick - тик

 TF - таймфрейм  (получить его можно с помощю  TF.fromSec(число_секунд)  или TF.fromName(имя)
 
 CBars - бары по одному таймфрейму. Хранит также размер тика, который можно задать либо вручную, либо будет вычислен автоматически.  Класс иммутабелен (не изменяется)
 
 CBarsMutable - изменяемый класс, можно добавлять бары или тики в конец.

Оба класса наследуются от абстрактного класса IBars 

 CQuotesHistory - класс истории по разным таймфреймам
 CQuotesHistoryMutable - изменяемый класс.  Можно добавлять бары и тики.

Запрос баров - метод Bars(tf : TF).  Полученный объект будет иммутабелен (никогда не изменится)
